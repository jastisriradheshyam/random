Hyperledger Fabric

Hyperledger Fabric Components

  • Organization CA ( Certificate Authority )
  • Peer
  • Ordering Service

Ordering Service

Responsibilities

  • What Organizations are participating in the network
  • Channels in the network
  • Organizations participation in the Channels
  • Policies enforcements in the network

Peer

Responsibilities

  • Transaction endorsement ( Endorser )
  • Maintaining Ledger ( Committer )
    • All peers are always committers.
  • Broadcasting events to interested parties
    • Block Commit event
    • Chaincode events

CA ( Certificate Authority )

Responsibilities

  • Managing network actors credentials
    • peer/node, orderer, user/transactor
  • Providing signed x.509 certificate
